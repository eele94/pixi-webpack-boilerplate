# pixi-webpack-boilerplate
## Boilerplate for pixi with webpack

Installing through npm
`npm install`

Als install webpack dev server
`npm install -g webpack-dev-server`

Start
`npm run start`

**PreLoaders**
- eslint

**Loaders**
- babel-loader
- css-loader
- sass/scss-loader
- less-loader
- url-loader(fonts and img)
